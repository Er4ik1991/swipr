СвайпДжоб — прототип приложения
================================

Это готовый проект. Запускать вручную ничего не нужно — просто
откройте эту папку в Claude Code и напишите ему обычными словами.

КАК ЗАПУСТИТЬ (через Claude Code)
---------------------------------
1. Установите и откройте приложение Claude (десктоп), вкладка Code.
2. Откройте в нём ЭТУ папку (swipejob).
3. Напишите Claude Code фразу:

   «Установи зависимости и запусти этот проект в режиме разработки»

   Он сам выполнит установку и запуск, а потом даст ссылку
   (обычно http://localhost:5173) — откройте её в браузере.

4. Дальше меняйте приложение, просто описывая словами, что хотите:
   например «добавь экран чата после совпадения».

ЧТО ГДЕ ЛЕЖИТ
-------------
- src/App.jsx — само приложение (карточки, свайпы, экран выбора роли).
- src/main.jsx — точка входа, трогать не нужно.
- index.html — страница-обёртка, трогать не нужно.
- package.json — список того, что нужно установить.

Если что-то не запускается — просто скопируйте текст ошибки
и вставьте его в Claude Code, он подскажет, что делать.
